# Morse Trainer

**Morse Trainer** — многофункциональный онлайн-тренажёр азбуки Морзе, созданный в рамках проекта  
[arduino-nano-morse-keyboard](https://github.com/entitybtw/arduino-nano-morse-keyboard)  
и для участия в VIII региональном конкурсе исследовательских и проектных работ школьников  
**«Высший пилотаж — Пенза 2026»**.

Веб-версия: **https://morse.entitybtw.ru**

---

## Возможности

- Система рекордов  
- Режимы тренировки  
  - ввод с клавиатуры  
  - ввод кодом Морзе  
- Поддержка языков  
  - русский  
  - английский  
- Встроенная шпаргалка азбуки Морзе
